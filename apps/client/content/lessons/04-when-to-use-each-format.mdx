# When to Use Each Format

Knowing *how* to transpose is only half the skill. Knowing *when* to transpose — and which format a given context requires — is equally important.

## Minus Cylinder: When to Use It

Use minus cylinder when:

- **Recording prescriptions** from an ophthalmologist or when documenting in a minus-cylinder practice's records
- **Entering data into a minus-cylinder phoropter** for refraction or verification
- **Reading from a lensometer** set to minus cylinder mode
- **Communicating with an MD** who writes in minus cylinder convention

Minus cylinder is dominant in medical ophthalmology. If you receive a handwritten prescription without labels, and it was written by an ophthalmologist, assume minus cylinder.

## Plus Cylinder: When to Use It

Use plus cylinder when:

- **Submitting to an optical laboratory** — most surfacing labs work in plus cylinder
- **Entering prescriptions into auto-refractor or keratometry software** — these instruments typically output plus cylinder
- **Working with a plus-cylinder optometrist** — confirm their preference at the start
- **Ordering contact lenses** — contact lens parameters are often specified in plus cylinder

## Verification by Back-Transposition

After transposing a prescription, verify your work by transposing the result again. Applying the transposition formula twice returns you to the original prescription. If it does not, an error was made.

Example:
```
Original:     -2.00 -1.00 x 180
Transposed:   -3.00 +1.00 x 090
Back-check:   -2.00 -1.00 x 180  ← must match original
```

This back-transposition check takes ten seconds and catches every arithmetic error. Build it into your routine whenever accuracy is critical.

## Building the Habit

In a busy clinical environment, transposition must be automatic. The three rules — new sphere, flip cylinder sign, shift axis by 90 — should be as reflexive as checking a patient's name before calling them. The practice mode in this app exists specifically to build that automaticity through active recall.
